This data set has been created from scraping the official website of Ambition Box
(i.e. https://www.ambitionbox.com/list-of-companies?campaign=desktop_nav&page=1 ) 
which contains the data of companies, including information like, Name of the Company,
Location of the Company, Type of Company and Rating of the Company.

The size of the dataset is (1180, 4).

This data set may contain some discrepancies, as it has not been cleaned and has been directly published on this website.
